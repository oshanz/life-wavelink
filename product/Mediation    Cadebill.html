Data mediation

Cadebill mediation engine provides a comprehensive set of tools that allows you to:

  • Collect and validate CDRs from any upstream source, in any format

  • Correlate and segment different input source CDRs

  • Aggregate of partial CDRs related to one master CDR

  • Convert raw CDR records with information such as price, cost, Quality of Service (QoS) parameters

  • Transform data to different output streams/formats such as XML, text, CSV for use by downstream external processes

  • Transform mediation data with user defined templates, user defined rules

  • Provide information needed for revenue assurance, vendor auditing, fraud management and business intelligence engine

  • Standardize, archive usage data to generate custom reports and perform other functions for use by downstream external processes
