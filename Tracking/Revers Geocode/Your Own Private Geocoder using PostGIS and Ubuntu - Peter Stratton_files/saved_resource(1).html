          I solved the junk in sh script problem by running the generate function from the command line with psql. The -At option eliminates "+" symbols and the column header from the output. So the command to generate the nation data loader script would be:

          psql -At -U postgres -c "SELECT loader_generate_nation_script('ubuntu');" > tiger_nation.sh

          Hope it helps!
