    Hi, just plug the usb modem straight into your pc. Start the dashboard software that comes with the modem ( the utility that that auto installs - Mobile Partner or what it is called) one open and sim card was detected, go to Tools then Options. In the Options Tab, go to Network and then Network type. On the drop down select WCDMA only, click ok and close Mobile Partner app. The setting is now saved on the modems config. Plug the usb back into the tp-link router and the router will now connect only via 3g.

    You can reverse this again by plugging the usb back into the pc and go to option ....... And then select LTE only or Auto. Hope this helps.

    I can only confirm that i have the same issue with E398 and TP-MR3420 V2. If it is like some of you say, 3G works but not 4G it is probably a firmware/software bug, because in E398 3G works like a serial connection and 4G is NDIS. The internal firewall in TP-MR3420 is probably listen on serial instead of NDIS. I have tried this with the modem connected to my Dovado 4GR in bridged mode and then connect to the WAN port and it works perfect, but when i connected E398 direct into 3420,s USB it won´t work, i can surf normally but no open ports.

    Sad but true, TP-Link has to write a new firmware to fix this

    /Mats

    TPLINK have been very slow in implementing full support for 4G/LTE modems.

    Huawei 4G modems have a different response to mode and band commands.

    You will also find, if you are in an LTE location that gives speeds greater than 25Mbps, that the V2 will (most likely) hard limit at 25Mbps.

    The Tiny will support speeds up to 88Mbps (lan connection).

    It is pretty bad form, that TPLINK have released a 3/4G modem ... that does 4G so poorly!!
